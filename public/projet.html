À propos du projet

snowcheck.ch — conditions de ski en temps réel pour les stations Magic Pass en Suisse

Pourquoi ce site ?

Choisir où skier un week-end n'est pas anodin : il faut croiser la météo, l'enneigement, les risques d'avalanche, le temps de route et les foules potentielles. Faire cette synthèse manuellement, station par station, prend du temps.

snowcheck.ch agrège automatiquement ces données pour les 92 stations du réseau Magic Pass et les classe selon un score unique, personnalisé en fonction de votre point de départ en Suisse.

Le score et les verdicts

Chaque station reçoit un score de 0 à 100 calculé à partir de sept critères pondérés : ensoleillement, neige fraîche, hauteur de neige, pourcentage de pistes ouvertes, indice Jour Blanc, vent et fréquentation estimée. Lorsqu'une localisation est renseignée, un bonus de proximité est ajouté pour favoriser les stations accessibles.

Le score se traduit en quatre verdicts :

Excellent — ≥ 70 Bon — ≥ 45 Correct — ≥ 20 Difficile — < 20

En cliquant sur une carte de station, on accède au détail complet : prévisions jour par jour, décomposition du score par critère, lien webcam et plan des pistes.

Contexte du projet

Ce site est un projet personnel, développé sans expérience préalable en développement web. L'ensemble du code — frontend React, fonctions serverless Cloudflare, pipeline CI/CD, tests unitaires — a été produit par dialogue itératif avec Claude Code (Anthropic), un assistant de développement par IA.

L'objectif était de démontrer qu'un profil non-développeur (ICT Manager, diplôme fédéral en informatique de gestion) peut concevoir et mettre en production une application web complète en pilotant efficacement un outil d'IA générative. Le code source est public et documenté.

snowcheck.ch est un projet indépendant, non affilié à Magic Pass SA. Les données proviennent de sources publiques et gratuites dont la disponibilité n'est pas garantie. Le site est destiné à un usage personnel et éducatif.
